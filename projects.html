MovieApp

MoviesNow, crafted with React.js, Flask, and CSS, stands out with its distinctive features in the realm of streaming platforms. Users enjoy a precise search function for movies and TV shows, refined filtering by genre, and dynamic carousels showcasing weekly trending content. The homepage offers a snapshot of the latest trends, while analytics tools track top-rated movies, all-time highest grossers, and the popularity of genres, providing a comprehensive and engaging cinematic experience.

Key Features

-MoviesNow offers several exciting features that set it apart from other streaming websites.
-Users can search for specific movies or TV shows using our search feature, which provides accurate results based on keywords.
-filtering feature allows users to refine their search results by genre.
-Navigation to relevant videos of the title.
-Dynamic Carousel for trending movies and tv shows for every week.
-Home page displays trending movies and tv shows per week.
-incorporated analytics features that enable us to track top Rated Movies trending this week, Highest grossing movies of all time, and Proportion of popular Genres.

Covid Dashboard

The COVID Dashboard is a Vue.js-powered web application designed to provide real-time statistics on the global impact of COVID-19. It offers users an intuitive and dynamic interface to explore and comprehend the latest information related to the pandemic. The dashboard includes overall statistics, country-specific data, and interactive charts, enabling users to stay informed about the current state of the pandemic worldwide.
Tools Used

-Vue.js: The core framework for building the user interface and handling dynamic components.
-Axios: A lightweight HTTP client used for making API requests to fetch real-time COVID-19 data.
-COVID-19 Data API: An external API serving real-time COVID-19 statistics, providing the necessary data for the dashboard.
-Chart.js: A charting library that can be integrated to visualize data through interactive and aesthetically pleasing charts.

Data Visualization using Tableau

The Data Visualization project, "Exploring the Impact of Inflation on Consumers and the Economy," leverages Tableau to deliver valuable insights into the effects of inflation on both individual consumers and the broader economic landscape. Through the visualization of data collected from diverse sources, the project aims to uncover patterns, trends, and correlations, providing a comprehensive understanding of how inflation influences various economic indicators.
Tableau: The primary tool for creating interactive and insightful data visualizations. Tableau enables the exploration and analysis of complex datasets, making it ideal for showcasing the impact of inflation on different facets of the economy.
